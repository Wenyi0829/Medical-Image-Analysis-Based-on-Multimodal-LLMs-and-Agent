# Medical VQA + Agent Fine-tuning

This repository is a work-in-progress project for fine-tuning a vision-language model on medical Visual Question Answering (VQA) with tool‑augmented agent capabilities.

## Data & Model

Dataset: LLaVA‑Med (10K instructions + images)
url{https://github.com/microsoft/LLaVA-Med}

Base Model: Qwen3‑VL (8B variant with thinking)
url{https://github.com/QwenLM/Qwen3-VL}

The temporary results are stored in eval_results/.
